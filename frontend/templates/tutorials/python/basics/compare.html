Операторы сравнения

Операторы сравнения используются в Python для сравнения значений. Результатом сравнения является логическое значение True или False.

Оператор Описание Пример Результат
== Равно 2 == 2 True
!= Не равно 2 != 3 True
< Меньше 2 < 3 True
> Больше 3 > 2 True
<= Меньше или равно 2 <= 3 True
>= Больше или равно 3 >= 2 True
