Операторы и выражения в Python

Операторы используются в Python для выполнения различных операций, таких как арифметические вычисления, сравнения, присваивания и т.д. В этом уроке мы рассмотрим различные операторы и выражения в Python.

Арифметические операторы

Арифметические операторы используются в Python для выполнения различных арифметических вычислений, таких как сложение, вычитание, умножение и т.д.

Оператор Описание Пример Результат
+ Сложение 2 + 2 4
- Вычитание 5 - 3 2
* Умножение 2 * 3 6
/ Деление 6 / 3 2
// Целочисленное деление 7 // 3 2
% Остаток от деления 7 % 3 1
** Возведение в степень 2 ** 3 8

Операторы присваивания

Операторы присваивания используются для присваивания значений переменным.

Оператор Описание Пример
= Присваивание значения x = 10
+= Прибавление значения x += 5 (эквивалентно x = x + 5)
-= Вычитание значения x -= 5 (эквивалентно x = x - 5)
*= Умножение на значение x *= 5 (эквивалентно x = x * 5)
/= Деление на значение x /= 5 (эквивалентно x = x / 5)
//= Целочисленное деление на значение x //= 5 (эквивалентно x = x // 5)
%= Остаток от деления на значение x %= 5 (эквивалентно x = x % 5)
**= Возведение в степень на значение x **= 5 (эквивалентно x = x ** 5)

Операторы сравнения

Операторы сравнения используются в Python для сравнения значений. Результатом сравнения является логическое значение True или False.

Оператор Описание Пример Результат
== Равно 2 == 2 True
!= Не равно 2 != 3 True
< Меньше 2 < 3 True
> Больше 3 > 2 True
<= Меньше или равно 2 <= 3 True
>= Больше или равно 3 >= 2 True
