Операторы условий в Python

Операторы условий используются в языке Python для выполнения различных действий в зависимости от выполнения условий. Они позволяют программисту создавать более гибкие и управляемые программы, которые могут изменять свое поведение в зависимости от различных факторов.

Оператор if

Оператор if используется для выполнения блока кода, если определенное условие истинно.

if условие:
		блок кода

Оператор if-else

Оператор if-else используется для выполнения блока кода, если определенное условие истинно, и другого блока кода, если условие ложно.

if условие:
		блок кода1
	else:
		блок кода2

Оператор if-elif-else

Оператор if-elif-else используется для выполнения блока кода, соответствующего первому истинному условию, или блока кода, связанного с последним блоком else, если все условия ложны.

if условие1:
		блок кода1
	elif условие2:
		блок кода2
	else:
		блок кода3
