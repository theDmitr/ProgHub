Логические операторы в Python

Логические операторы используются в языке Python для объединения нескольких условий в одно. Они позволяют программисту создавать более гибкие и управляемые программы, которые могут изменять свое поведение в зависимости от различных факторов. В Python есть три логических оператора: and, or и not.

Оператор and

Оператор and объединяет два условия и возвращает True, если оба условия истинны. В противном случае он возвращает False.

x = 5
    y = 10
    if x > 0 and y > 0:
        print("x and y are both positive")

Оператор or

Оператор or объединяет два условия и возвращает True, если хотя бы одно из условий истинно. В противном случае он возвращает False.

x = 5
    y = -10
    if x > 0 or y > 0:
        print("at least one of x and y is positive")

Оператор not

Оператор not инвертирует значение условия. Если условие истинно, то оператор not возвращает False. Если условие ложно, то оператор not возвращает True.

x = 5
    if not x > 0:
        print("x is not positive")

Приоритет операторов

При использовании логических операторов очень важно помнить о порядке выполнения операций. В Python оператор not имеет высший приоритет, затем идет оператор and, а затем оператор or. Если вам нужно изменить порядок выполнения операций, то можно использовать скобки.

x = 5
    y = 10
    z = 15
    if (x > 0 and y > 0) or z > 0:
        print("at least one of x, y, and z is positive")

Заключение

Они позволяют программисту создавать более гибкие и управляемые программы, которые могут изменять свое поведение в зависимости от различных факторов.
