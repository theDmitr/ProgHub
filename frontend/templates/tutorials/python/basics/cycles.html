Операторы цикла в Python

Операторы цикла используются в языке Python для повторения одного и того же блока кода несколько раз. Это позволяет программисту создавать более эффективные и управляемые программы.

Оператор while

Оператор while повторяет блок кода, пока условие истинно. Когда условие становится ложным, выполнение кода выходит из цикла. Например:

i = 1
    while i <= 10:
        print(i)
        i += 1

В этом примере мы используем оператор while, чтобы повторить вывод чисел от 1 до 10. Переменная i увеличивается на 1 на каждой итерации, пока не достигнет значения 11, при котором условие становится ложным и выполнение цикла прекращается.

Оператор for

Оператор for используется для перебора элементов в последовательности (такой как список, кортеж или строка). Он повторяет блок кода для каждого элемента в последовательности. Например:

fruits = ["apple", "banana", "cherry"]
    for fruit in fruits:
        print(fruit)

В этом примере мы используем оператор for, чтобы повторить вывод элементов списка fruits. На каждой итерации переменная fruit принимает значение следующего элемента списка.

Оператор break

Оператор break используется для прерывания выполнения цикла. Когда он достигается, выполнение кода выходит из цикла. Например:

i = 1
    while i <= 10:
        if i == 5:
            break
        print(i)
        i += 1

В этом примере мы используем оператор break, чтобы прервать выполнение цикла, когда переменная i достигнет значения 5. В результате будут выведены только числа от 1 до 4.

Оператор continue

Оператор continue используется для перехода на следующую итерацию цикла, минуя оставшуюся часть блока кода. Например:

i = 1
    while i <= 10:
        if i == 5:
            i += 1
            continue
        print(i)
        i += 1

В этом примере мы используем оператор continue, чтобы пропустить вывод числа 5. В результате будут выведены числа от 1 до 4, а затем от 6 до 10.

Заключение

Операторы цикла являются важной частью языка Python и используются во многих программах. Они позволяют программисту создавать более эффективные и управляемые программы. Надеюсь, эта статья поможет вам лучше понять, как использовать операторы цикла в ваших программах на Python.
